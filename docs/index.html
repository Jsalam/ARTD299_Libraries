getQuad

Gets four contiguous elements from the one dimensional array. The anchor element is at the col and row intersection. The remaining three are on the right, right-lower, and lower of the anchor.

getQuad(arr: array, col: number, row: number, matrixWidth: number): any
Parameters
arr (array) the one-dimensional array
col (number) the column index
row (number) the row index
matrixWidth (number) the matrix row lenght
Returns
any: An array with elements orders as follows: anchor, right, lower-right, lower
